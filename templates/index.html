How to Use This System

This advanced system analyzes Synthetic Aperture Radar (SAR) imagery from NASA satellites to assess 9 types of climate disaster risks: floods, urban heat islands, forest fires, deforestation, droughts, tsunamis, landslides, cyclones/hurricanes, and volcanic eruptions.

Input Options
  • SAR Imagery: Upload TIFF, HDF5, NC4, BSQ, FILT, SHP, JPG, or PNG files
  • Manual Features: Enter {{ model_info.n_features if model_info else '15' }} comma-separated values
  • Data Type: Select disaster category for optimized analysis
Feature Extraction
  • Statistical: Mean, std, min, max, median intensity
  • Texture: GLCM homogeneity, energy, entropy, contrast
  • Advanced: Gradient, FFT, domain-specific indices

Example Manual Input: 156.5,45.2,89.1,255.0,178.3,0.85,0.92,2.1,8.5,12.4,8.9,1250.5,890.2,0.65,0.75
